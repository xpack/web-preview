
## The xPack preview site

A placeholder to experiment with new versions of the xPack web site.

The actual content is not here, but in the separate `gh-pages` branch.

By default it is disabled (this content is shown); to make it visible
change the **Settings** -> **GitHub Pages** selection from 
`master branch` to `gh-pages branch`.

The preview URL is: https://xpack.github.io/preview/

The GitHub project URL is: https://github.com/xpack/preview
